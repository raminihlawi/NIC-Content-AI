# NIC Visual Guide For Image Prompts

Denna guide används när bildprompter tas fram för NIC.

## Visuell inspiration från nord-ic.com

Baserat på budskapet och uttrycket på `https://www.nord-ic.com/` bör bilderna kännas:

- ljusa
- tekniska
- rena
- strukturerade
- sakliga
- moderna

Det visuella ska förstärka:

- automatiserad regelefterlevnad
- datadriven styrning
- informationsklassificering
- policyefterlevnad
- riskreduktion
- kontroll över ostrukturerad information

## Rekommenderad stil

- clean editorial tech illustration eller stilren 3D-render
- premium B2B-uttryck
- skandinavisk enkelhet
- luftiga ytor
- tydlig komposition
- fokus på informationsflöden, dashboards, datalager, klassificering och kontroll

## Färgpalett

- ljus bakgrund
- blå
- teal
- kall grå
- vita ytor
- lågmäld kontrastfärg vid behov

## Undvik

- mörka hacker-miljöer
- hoodies, gröna kodregn och lås-ikoner som huvudmotiv
- överlastade dashboards
- generiska stock-photo-team i konferensrum
- för mycket text i bilden
- dramatisk cyberpanik

## Bra motiv för NIC

- strukturerade informationsflöden
- dokumentmiljöer med tydlig klassificering
- policykontroll i dashboard-form
- data som går från kaos till struktur
- ledningsnära visualisering av risk och efterlevnad

## Basrad för promptar

Använd gärna denna grund i bildprompter:

`Create a clean, modern LinkedIn image for a regulatory compliance and information security company. Style: minimal editorial tech illustration, premium Scandinavian feel, bright background, subtle blue, teal and soft gray palette, high clarity, clean composition, no clutter, no watermark, no text in the image.`
